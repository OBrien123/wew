Présentation de la Licence 1 de la L2i

Au cours de la première année, les étudiants acquièrent les bases essentielles de l'informatique, y compris les concepts de programmation, les algorithmes de base, les mathématiques appliquées à l'informatique et les bases de données.
Ce tableau resume les enseignement dont vous aurez besoin

Nom Description
Introduction à l'informatique Ce cours offre une introduction aux concepts de base de l'informatique, y compris l'histoire de l'informatique, les principes de base de la programmation et les fondamentaux des systèmes informatiques
Algorithmique et structures de données Les étudiants apprennent les principes fondamentaux de l'algorithmique et de la conception de structures de données. Cela inclut l'apprentissage des algorithmes de tri, de recherche, et des structures de données telles que les listes, les arbres et les graphes
Mathématiques pour l'informatique Ce cours couvre les bases mathématiques nécessaires à la compréhension de l'informatique, y compris les fonctions, les ensembles, les relations, les probabilités et les statistiques
Introduction aux langages de programmation Les étudiants sont introduits à plusieurs langages de programmation, tels que Python, Java ou C, et apprennent les concepts de base de la programmation tels que les variables, les boucles et les fonctions

Présentation de la Licence 2 de la L2i

Au cours de la deuxième année, les étudiants approfondissent leurs connaissances en informatique avec des cours plus avancés et des projets pratiques.
Ce tableau resume les enseignement dont vous aurez besoin

Nom Description
Programmation avancée Ce cours approfondit les concepts de programmation, en abordant des sujets tels que la programmation orientée objet, les structures de contrôle avancées et les bibliothèques logicielles
Systèmes d'exploitation Les étudiants apprennent les principes de base des systèmes d'exploitation, y compris la gestion des processus, la gestion de la mémoire et les systèmes de fichiers
Réseaux informatiques Ce cours couvre les principes fondamentaux des réseaux informatiques, y compris les protocoles de communication, les architectures réseau, la sécurité réseau et les technologies de réseau sans fil
Bases de données Ce cours aborde les concepts de base des bases de données, y compris la modélisation des données, le langage SQL et les opérations de base de données telles que l'insertion, la mise à jour et la suppression de données

Présentation de la Licence 3 de la L2i

La troisième année permet aux étudiants de consolider leurs connaissances et leurs compétences en informatique en se concentrant sur des domaines spécifiques tels que le développement web, la sécurité informatique et le développement d'applications mobiles. Ils réalisent également un projet de fin d'études où ils appliquent leurs connaissances pour résoudre un problème pratique ou développer une application informatique. En option, les étudiants peuvent choisir de réaliser un stage en entreprise pour acquérir une expérience pratique dans un environnement professionnel.
Ce tableau resume les enseignement dont vous aurez besoin

Nom Description
Développement web Ce cours se concentre sur le développement d'applications web, y compris les langages de programmation web (comme HTML, CSS et JavaScript), les frameworks web et les technologies de base de données associées
Sécurité informatique Les étudiants approfondissent leurs connaissances en sécurité informatique, en abordant des sujets tels que la cryptographie, la sécurité des réseaux, la sécurité des applications et la gestion des incidents de sécurité
Projet de fin d'études Les étudiants réalisent généralement un projet de fin d'études, où ils appliquent les connaissances et les compétences acquises tout au long de leur formation pour résoudre un problème pratique ou développer une application informatique
Stage en entreprise Ce programme permet les étudiants d'avoir l'opportunité d'acquérir une expérience pratique dans un environnement professionnel et d'appliquer leurs compétences en situation réelle
